About Project

In this project, we've created a web-based application using Django, HTML, and CSS to analyze comments from various social media platforms. Leveraging APIs like "tweepy" for Twitter and "googleapiclient" for YouTube, our application performs sentiment analysis on user posts. By categorizing comments as positive, negative, or neutral, it offers valuable insights for content creators aiming to better understand their audience's preferences. This tool simplifies the otherwise cumbersome task of manually analyzing comments, making it user-friendly and personalized for each individual.

Technology Used

  • Front-End: HTML 5.0, CSS 3.0
  • Back-End: Django, Python
  • Integrated Development Environment : VS-Code
  • Library: Tweepy, NLTK, googleapiclient

About Project

In this project, we've created a web-based application using Django, HTML, and CSS to predict the likelihood of heart disease and future risk based on user-entered data. Leveraging regression analysis, including linear regression for percentage predictions and logistic regression for binary classification, our application offers real-time assessments. It empowers users to make informed decisions about their heart health, featuring graphical comparisons, training data, and testing data. Given the rising incidence of heart issues, this tool serves as a valuable resource for individuals of all ages seeking to monitor and manage their heart health.

Technology Used

  • Front-End: HTML 5.0, CSS 3.0
  • Back-End: Django, Python
  • Integrated Development Environment : VS-Code
  • Data Source: Kaggle.com
  • Library : Sklearn, Pandas, Matplotlib, Numpy, Random
  • Data Visualization : Seaborn

About Project

In this project, we've created a web-based movie rating prediction application using Django, HTML, and CSS. We explored and compared four regression algorithms: Linear Regression, Decision Tree Regressor, K-neighbors Regressor, and Random Forest Regressor to determine the most accurate predictor. Our user-friendly interface allows users to effortlessly predict movie ratings by inputting basic details, while also providing insight into the underlying analysis, including data visualization and model comparisons. This tool empowers movie enthusiasts and industry professionals to make informed decisions regarding film ratings.

Technology Used

  • Front-End: HTML 5.0, CSS 3.0
  • Back-End: Django, Python
  • Integrated Development Environment : VS-Code
  • Database: Pandas
  • Machine Learning Library: Sklearn

ANALYSIS OF TWITTER REVIEWS ON COVID-19

Abstract

Currently, a newly born genome COVID declared as pandemic has spread globally and led to thousands of deaths, having an enormous impact on health, systems, and economies. COVID pathogenic viral infection is seeking attention worldwide, whether forcefully or being concerned. In such high social pacing, social media platforms have emerged as an alternate way for people to express their thoughts and reviews. TWITTER is one of the most popular platforms used worldwide by people to express their opinions. Using TWITTER API or Beautiful soul the latest reviews and comments related to COVID-19 will be extracted and analyzed. The motivation behind this review and analysis is to get feedback about how people think about the current situation using their positive or negative sentiment. Analysis has been done covering various aspects on the basis of economic conditions. Data mining tools can easily resolve those issues that traditionally were too time-consuming. Data mining of social networks can be done using graph mining methods such as classification, prediction, detection, pattern measurement, efficiency, and modeling, which are the basic part of data mining.
The global knowledge base should be strengthened for evidence-based decision-making in preventing and addressing the COVID-19 pandemic issues.
